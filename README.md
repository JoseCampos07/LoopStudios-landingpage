# 🏝️ Proyecto: Loopstudios Landing Page

Este proyecto consiste en el desarrollo de la **landing page de Loopstudios** utilizando **Astro** y **Tailwind CSS**.  
El objetivo es aplicar los conocimientos sobre **componentes de Astro**, **maquetación**, **estilos responsivos** y **utilidades CSS** para construir un diseño limpio, moderno y adaptable a diferentes dispositivos.

### 💡 Lo que aprendí

Gracias a este proyecto, he puesto en prácica de manera más masiva la programación en Astro y Tailwind, además de un poco de Javascript para el menú desplegable que se muestra en la versión móvil.
Tuve que investigar más conceptos sobre Astro y Tailwind para hacer que la página se asemejara lo mejor posible a los diseños originales que se proporcionaron, fue un trabajo largo, pero satisfactorio al ver finalmente el sitio funcionando de manera completa.
Además, he aprendido a usar de mejor manera los media queries y la clase hidden para ocultar elementos según el tamaño de pantalla donde se encuentren, para hacer la página más responsiva.
